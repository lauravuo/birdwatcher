# Birdwatcher

A mobile-first web application for birdwatchers to track and share their sightings within groups.

## Features

- **Authentication**: Sign in with Google.
- **Group Management**: Create or join birdwatching groups.
- **Recognitions**: Log bird sightings with month validation.
- **Statistics**: View monthly and yearly stats for yourself and your group.
- **Mobile-First Design**: Optimized for use in the field.

## Tech Stack

- **Framework**: React + Vite + TypeScript
- **Styling**: Vanilla CSS (Nature-inspired theme)
- **Backend**: Firebase (Auth, Firestore, Hosting)
- **Tooling**:
  - Linting/Formatting: Biome
  - Unit Testing: Vitest
  - E2E Testing: Playwright
  - CI/CD: GitHub Actions

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Start dev server: `npm run dev`
4.  Run tests: `npm run test` or `npm run test:e2e`
